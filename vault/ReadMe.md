# Casino Delight Knowledge Base

Welcome to your project's **Obsidian Vault**. This is your central hub for managing content, automation, and technical knowledge.

## 📂 Vault Structure

- **Blogs/**
    - **Drafts/**: Write your new articles here in Markdown.
    - **Published/**: Archive for articles that are already live in the app.
- **Technical Docs/**
    - **Supabase/**: Notes on database schema and storage.
    - **n8n/**: Workflows and automation logic.
    - **Flutter/**: UI components and branding.
- **Attachments/**: Images and media used in your notes.

## ⚙️ How to Use with n8n

1. **Write**: Create a new `.md` file in `Blogs/Drafts`.
2. **Draft**: Use standard Markdown (links, bold, headings).
3. **Automate**: Link your n8n workflow to read these files (or use your Excel-to-n8n pipeline).
