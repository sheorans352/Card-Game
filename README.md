# Card Game

A Flutter project demonstrating integration with Supabase and Riverpod.

## Technical Stack

- **Flutter**: UI Framework
- **Supabase**: Backend-as-a-Service (Database, Auth, etc.)
- **Riverpod**: State Management

## Getting Started

To run this application, make sure you have Flutter installed. Then run:

```bash
flutter pub get
flutter run
```

This app contains a simple `main.dart` file that initializes Supabase with the provided URL and Anon Key, and verifies the connection to the database.
